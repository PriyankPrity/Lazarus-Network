About Sotreus

Lazarus Dedicated Network is an encrypted tunneling solution for end users offering WireGuardÂ® Tunneling, secured by Ed25519 Private Key Encryption, Multi Region Network, Peer to Peer network access and a lot more.

This service offers a dedicated IP based virtual private network to the users for connecting their enterprise or home smart devices so that they can be accessed from anywhere.

It is focused on enterprise grade security level configurations and comes integrated with a Firewall in the same region as the VPN.

All the configurations are logged and managed through the organization owners and admins.

It supports higher bandwidth and optimized performance for enterprise software usages requiring good latency. The devices are now within another layer of security as firewall rules and permissions can be set over them.

With the VPN it's easy to connect, communicate and control the smart devices remotely.
